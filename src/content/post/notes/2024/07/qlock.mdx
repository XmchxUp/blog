使用二进制来当成 5 \* 3 的数字展示，例如

```python
>>> bin(31599)
'0b111101101101111'


111
101
101
101
111

这就是0
```

由于 go 代码需要严格的格式化，则暂时不知道有啥办法，只能使用 tsoding 提供的 std_c_lexer.h 来 format go 代码当成文件 embed 到的代码中。

带有颜色的输出

```go
fmt.Printf("\033[1;41;30m%c\033[0m", ch)
```

当每次输出完，屏幕有个 cursor 指针，则移动这个 cursor 到输出前的位置，则在下一次时候完成刷新屏幕。

```go
// Print the escape sequences to move the cursor up and left
fmt.Printf("\033[%dA\033[%dD", h, w)
```
